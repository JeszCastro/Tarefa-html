Perguntas e Respostas

  1. Na tag "input"com atributo type="text", para que serve o atributo "placeholder"?

    serve para exibir um texto de exemplo dentro da caixa de texto antes do usuário inserir qualquer informação.

  2. Na tag "input" com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    serve para indicar que uma opção ou caixa de seleção está pré-selecionada por padrão.

  3. Na tag "input" para que serve o atributo "value"?

    é usado para definir o valor padrão que será enviado para o servidor quando um formulário é enviado.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    O atributo "name" deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez

  5. Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?

    A tag "label" é usada para associar um rótulo descritivo a um elemento de formulário, como um campo de entrada ou botão. O atributo "for" na tag "label" é usado para indicar qual elemento do formulário o rótulo está associado.

  6. Na tag "option" para que serve o atributo "selected"?

    é usado para indicar qual opção de uma lista suspensa deve ser exibida como selecionada por padrão.

  7. Nas tags de formulário para que serve o atributo "required"?

    O atributo "required" nas tags de formulário é usado para especificar que um campo de entrada é obrigatório e deve ser preenchido pelo usuário antes que o formulário possa ser enviado.

  8. Para que serve a tag "textarea"?

    é usada para criar uma área de texto expansível em um formulário HTML, onde o usuário pode inserir texto em várias linhas.

  9. Na tag "form" para que serve o atributo "action"?

    é usado para especificar a URL do servidor que receberá os dados do formulário quando ele for enviado.

  10. Na tag "form" para que serve o atributo "method"?

    é usado para especificar como os dados do formulário serão enviados para o servidor. Existem dois valores possíveis para esse atributo: "get" e "post".

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?

    O valor do atributo "method" na tag "form" não tem relação com a exibição de senhas na barra de endereços do navegador. Para evitar que a senha seja exibida na barra de endereços, é necessário usar o atributo "type" com o valor "password" na tag "input" correspondente ao campo de senha.

  12. Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?

    O valor do atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos deve ser definido como "post". Isso ocorre porque o método "post" permite que os dados do arquivo sejam enviados no corpo da solicitação HTTP, o que é necessário para o upload de arquivos.

  13. O que as tags "fieldset" e "legend" produzem?

    são usadas em conjunto para criar um grupo de elementos relacionados em um formulário e fornecer uma legenda ou rótulo para esse grupo.
